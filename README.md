Catálogo de Cursos - Caso de Estudio (Opción #2)
1. Descripción del Proyecto

Este proyecto consiste en un artefacto simple, una mini app creada como parte de la actividad de coordinación y seguimiento del Caso de Análisis y Diseño de Sistemas. El objetivo principal de esta aplicación de consola es simular un proyecto técnico que permita practicar la asignación de tareas, la generación de evidencia mediante commits o capturas y el cumplimiento de los criterios Definition of Done (DoD) dentro de Trello.

En cuanto a su funcionalidad, el programa compila y se ejecuta correctamente en .NET. La aplicación muestra una lista estática de tres cursos y permite al usuario ingresar un ID, devolviendo el nombre y el área del curso correspondiente. Esto simula un filtro o búsqueda básica dentro del sistema.

2. Ejecución

La aplicación fue desarrollada en C# como un proyecto de consola de .NET. Para poder ejecutarla, es necesario contar con el SDK de .NET instalado en el equipo, recomendándose la versión 6.0 o superior.

Para iniciar el programa, primero se debe clonar el repositorio de GitHub donde se encuentra el proyecto. Una vez clonado, se debe abrir la consola (CMD o PowerShell) y navegar hasta la carpeta que contiene el archivo .csproj del proyecto. Desde allí, se ejecuta la aplicación utilizando el comando dotnet run.

Al iniciar, el programa solicitará al usuario que ingrese un ID de curso. Una vez ingresado un número (1, 2 o 3) y presionada la tecla Enter, el programa mostrará el nombre y el área correspondiente al curso seleccionado. Este procedimiento permite simular una búsqueda o filtrado básico dentro de la mini app, cumpliendo con la funcionalidad principal del proyecto.

3. Evidencia y Coordinación

La gestión del proyecto se realizó utilizando un tablero Kanban en Trello, lo que permitió desglosar las tareas, darles seguimiento y mantener un registro organizado de los avances. Este tablero también sirvió para documentar la evidencia de cada actividad realizada, asegurando que se cumplieran los criterios de finalización establecidos.
