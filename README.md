Descripción del Proyecto
Este proyecto consiste en un artefacto simple, una mini aplicación web creada como parte de la actividad de coordinación y seguimiento del Caso de Análisis y Diseño de Sistemas. El objetivo principal de esta aplicación es simular un proyecto técnico que permita practicar la asignación de tareas, la generación de evidencia mediante commits o capturas y el cumplimiento de los criterios Definition of Done (DoD) dentro de Trello.

En cuanto a su funcionalidad, la aplicación está desarrollada en HTML, CSS y JavaScript y se ejecuta completamente en el navegador. La interfaz muestra una lista estática de tres cursos (ID: 1, 2, 3) y permite al usuario ingresar un ID en un campo de texto para activar una búsqueda, devolviendo el nombre y el área del curso correspondiente de forma dinámica en la pantalla. Esto simula un filtro o búsqueda básica dentro del sistema con un enfoque front-end.

Ejecución
La aplicación fue desarrollada utilizando HTML5, CSS3 y JavaScript puro. Para poder utilizarla, simplemente se necesita un navegador web moderno (como Chrome, Firefox, Edge, etc.).

Para iniciar el programa:

Se debe clonar el repositorio de GitHub donde se encuentra el proyecto.

Una vez clonado, simplemente se abre el archivo index.html (o el nombre que tenga el archivo HTML principal) directamente con el navegador o a través de un servidor local simple.

Al abrir la página, el usuario puede ingresar un ID de curso (1, 2 o 3) en el campo de texto y hacer clic en el botón "Buscar Curso". El programa mostrará dinámicamente el nombre y el área correspondiente al curso seleccionado en la sección de resultados. Este procedimiento permite simular una búsqueda o filtrado básico dentro de la mini app, cumpliendo con la funcionalidad principal del proyecto.
