# Caso-de-Analisis-y-Diseno-de-Sistemas---PCD
Este entregable, el "Caso de Análisis y Diseño de Sistemas," se enfoca en la coordinación del equipo: asignación, seguimiento y plazos. La mini app sirve como un artefacto simple para practicar la generación de evidencias y aplicar la DoD (Definition of Done), asegurando la calidad del análisis y diseño producido.
